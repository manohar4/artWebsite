# Images Directory

## Signature Logo

Please add your signature image file here with the name: `signature.png`

The image should be:
- Transparent background (PNG format recommended)
- High resolution (at least 800px wide)
- The signature should be in a color that contrasts well with purple backgrounds (white, light blue, or light colors work best)

If you don't have the image ready, the website will automatically fall back to a text version of your name.

## File Location

Place your signature image at: `/public/images/signature.png`

The website will automatically use it in the Logo component.

## Decorative Pattern Overlay

Please add your decorative pattern image here with the name: `decorative-pattern.png`

The image will be used as a subtle background overlay on the purple homepage.

The image should be:
- Light colored pattern (will be inverted and filtered for purple background)
- PNG format with transparent or light background
- High resolution (at least 1920px wide recommended)
- The pattern will be automatically:
  - Positioned in top-right and bottom-left corners
  - Applied with opacity filters (5-10% opacity)
  - Inverted and filtered to work with purple background
  - Blended using mix-blend-overlay for subtle effect

## File Location

Place your decorative pattern image at: `/public/images/decorative-pattern.png`

The website will automatically use it as a background overlay in the Logo component.

## Hero Portrait Image

Please add your hero portrait image here with the name: `hero-portrait.jpg`

This is the main hero image that will be displayed prominently in the hero section of the homepage.

The image should be:
- High resolution (at least 1920px wide recommended)
- Portrait orientation (vertical) works best

- Well-lit and high quality
- The image will be automatically:
  - Displayed in a beautiful glass-morphism frame
  - Enhanced with subtle gradients and effects
  - Responsive across all devices
  - Animated with parallax scrolling effects
