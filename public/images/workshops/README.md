# Workshop Images

Place your mandala workshop images in this directory:

- `workshop-1.jpg` - Main workshop image (people drawing at cafe tables)
- `workshop-2.jpg` - Group workshop image (people at wooden table)
- `workshop-3.jpg` - Close-up mandala drawing image

**Recommended image specifications:**
- Format: JPG or PNG
- Size: 1200px - 2000px width
- Aspect ratio: 4:3 or 16:9 for main images, 1:1 for thumbnails
- File size: Optimized for web (under 500KB per image)

Once you add the images, the MandalaWorkshop component will automatically display them.
